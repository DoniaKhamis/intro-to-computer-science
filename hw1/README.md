# Homework 1: Variables, Input & Output 📚

## 📝 Description
This assignment focuses on basic mathematical operations and handling user input and output in C. The program takes numerical input from the user to compute five distinct formulas:

1. **Walking Distance:** Calculates the total distance traveled based on speed and time.
2. **Cell Replication:** Calculates the exponential growth of cells after 2 seconds.
3. **Follower Growth:** Projects the future number of social media followers based on a monthly growth rate.
4. **Acceleration:** Computes the average acceleration of a vehicle given its initial velocity, distance, and time.
5. **Number Rounding:** Rounds a user-provided real number to the nearest integer.

## 🧠 Concepts Applied
* Variables and Data Types 
* Standard Console I/O 
* Mathematical and Arithmetic Operations
