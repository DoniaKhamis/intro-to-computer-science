# Homework 2: Conditions and Loops

## 📝 Description
This assignment focuses on implementing control flow using conditional logic and iterative loops. The program serves as a mathematical intuition tool, providing a menu-driven interface to evaluate three specific functions:

1. **Multivariable Function:** Evaluates a piecewise function $f(X,Y)$ based on the relationship between two integers, utilizing both `if-else` and `switch` statements.
2. **Expected Value:** Calculates the expected value $E[X]$ by taking pairs of values and probabilities. The program continues to prompt for input until the sum of probabilities reaches 1.0, automatically adjusting the final probability if the sum exceeds 1.
3. **Ascending Base with Descending Power:** Implements a product series formula $P(x,r)$ where a real-valued base increases while the power decreases over $r$ iterations.

## 🧠 Concepts Applied
* Control Flow (`if-else` and `switch`)
* Loop structures for menu persistence and mathematical summations
* Input handling and validation with `scanf`
* Formatting output to specific decimal precisions
