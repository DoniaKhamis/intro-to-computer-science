# Homework 3: Arrays and Functions

## 📝 Description
This assignment emphasizes modular programming by dividing tasks into specialized functions and utilizing 1D arrays for data storage. The core components include:

1. **Array Input Utility:** A function to safely read a specific number of integers into an array.
2. **Polynomial Evaluation:** Evaluates a polynomial of degree $d$ with given coefficients across a range of values ($0$ to $p-1$). The results are stored in an array and calculated using modular arithmetic.
3. **Minimum Plus Convolution:** Implements a complex array operation that finds the minimum modular sum of elements from two different arrays where the sum of their indices equals a target index $k$.

## 🧠 Concepts Applied
* 1D Array manipulation and indexing
* Modular arithmetic (ensuring positive results for modulo operations)
* Function modularity and passing arrays as parameters
* Algorithmic optimization for nested calculations
