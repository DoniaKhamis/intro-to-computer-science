# Homework 4: Pure Recursion

## 📝 Description
This assignment is a comprehensive exercise in recursive problem-solving. **The use of iterative loops (`for`, `while`, `do-while`) was strictly forbidden.** The program includes:

1. **Palindrome Check:** A recursive function that determines if the first $k$ and last $k$ characters of a string form a palindrome.
2. **Divisor Finder:** Recursively identifies and prints all divisors of a positive integer in ascending order.
3. **Multi-Reverse:** A recursive process that reverses segments of a string of length $k$ starting from each character sequentially.
4. **Zigzag Validation:** Recursively checks if an array follows a zigzag pattern ($arr[0] \le arr[1] \ge arr[2] ...$).
5. **Zigzag Transformation:** A recursive algorithm that reorders an array into a "Special Zigzag" where even indices are ascending and odd indices are descending.

## 🧠 Concepts Applied
* Recursive base cases and induction steps
* String and array processing without iteration
* In-place memory modification using recursion
