# Homework 5: Master of Pointers

## 📝 Description
This assignment simulates a circular elimination game used to decide who picks music for a road trip. The program manages a list of players and eliminates them based on a "remove" count until only one winner remains.

**Strict Constraint:** The use of array subscript notation (`[]`) was completely prohibited. All array traversal and data manipulation were implemented using **pure pointer arithmetic**.

The final output reorders the original array such that the winner is at the first position, followed by the players in the reverse order of their elimination.

## 🧠 Concepts Applied
* Pointer arithmetic and dereferencing
* Circular logic implementation within linear memory
* String handling via pointers to pointers (`char**`)
* In-place array reordering
