# Homework 7: Advanced Data Structures

## 📝 Description
The final assignment focuses on algorithm efficiency and complex data structures. It contains three primary modules:

1. **Optimized Search:** Implements a binary search variation to find the first occurrence of a number in a sorted array with $O(\log n)$ time complexity.
2. **Linked List Matrices:** Represents mathematical matrices using an array of linked lists (one list per row). It implements matrix multiplication ($A \times B = C$) without using 2D arrays.
3. **Fast Fibonacci:** Calculates the $n$-th Fibonacci value in $O(\log n)$ time using matrix exponentiation, leveraging the previously implemented matrix multiplication logic.

## 🧠 Concepts Applied
* Linked Lists (Node management and traversal)
* Big-O complexity optimization ($O(\log n)$ vs $O(n)$)
* Struct-based data modeling
* Efficient matrix exponentiation
